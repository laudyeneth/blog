Bootstrap 4!

Dec 12, 2018

Diferenciasprincipales entre Bootstrap 3 y 4

Procesador CSS: Cambio de procesador de css que tiene bootstrap 4, en esta versión se ha optado exclusivamente por Sass ya que compila más rápido.

Soporte Internet Explorer 8: Bootstrap 4 no tiene soporte para internet explorer 8.

Mejora de la Grida: El cambio es sencillo, se ha pasado de los pixeles a medidas relativas utilizando la unidad “rem” para conseguirlo.Este cambio también influye en la media queries.

No tiene GLYPHICONS: El que no poseea esto puede ser una ventaja o desventaja, dependiendo del usuario, ya que muchos no los utilizában nunca, con lo que es un peso que se quita de la css.

JAVASCRIPT reescrito en ES6: Los diversos plugin de Bootstrap han pasado a JavasScript ES6 que tiene una sintaxis mucho más clara y simple para la creación de objetos y las herencias.

Nuevas clases: Bootstrap 4 incorpora nuevas clases para dar estilos a los botones, estilos para cabeceras, tablas.

Incluye tarjetas que sustituyen a well, panel y thumbnail. Es un contenedor flexible que tiene opciones de introducir cabecera pie y gran variedad de contenidos.

También incluyen nuevas clases (Utility classes) que nos ayudaran a controlar los margin y padding de los elementos de nuestra plantilla.

CMD

Si no queremos instalar bootstrap 4 entonces podemos usar el CMD, así:

<!-- Latest compiled and minified CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css">

<!-- jQuery library -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>

<!-- Popper JS -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.3/umd/popper.min.js"></script>

<!-- Latest compiled JavaScript -->
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.1.3/js/bootstrap.min.js"></script>
